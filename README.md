# Austere theme

A simple, mostly grayscale theme.

![i3 + kitty + Neovim](images/preview.png)

> [!WARNING]
> Under development. The exact palette is subject to change.

Available for:
* bat and other `.tmTheme`-based software (only bat was tested)
* delta
* Dunst
* fzf
* i3, i3bar, i3lock-color
* kitty
* Neovim
* Rofi
* tmux
